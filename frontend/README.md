# Dubai Luxury Properties - Frontend

A modern, luxury real estate web application built with React.js, featuring Dubai's finest properties.

## 🌟 Features

- **Modern Design**: Dubai-inspired luxury aesthetic with gold accents
- **Responsive**: Fully responsive design for all devices
- **Property Listings**: Browse and filter luxury properties
- **Authentication**: User login and registration
- **Interactive UI**: Smooth animations and hover effects

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:3000`

## 🌐 Pages

1. **Home** - Hero section with featured properties
2. **Properties** - Full property listings with filters

4. **Login** - User authentication
5. **Register** - New user registration

## 🔗 API Integration

The frontend is configured to connect to a backend API at `http://localhost:5000/api`

Update the proxy settings in `vite.config.js` if needed.
