# Dubai Luxury Real Estate Platform - PFE Project

This project is a real estate appointment automation platform developed as part of a Final Year Project (PFE). It features a conversational AI for booking, workflow orchestration, and a modern property catalog.

## Project Structure

- **`/frontend`**: React + Vite application for the public property catalog and user interface.
- **`/backend`**: Node.js + Express API handling property data, appointments, and agent management.
- **`/docs`**: Project documentation, including the Cahier des Charges, Scrum plan, and technological choices.

## Technologies Used

- **Frontend**: React, Vite, Leaflet (Maps), i18next (Multilingual).
- **Backend**: Node.js, Express, PostgreSQL.
- **Tools**: Git/GitHub, Telegram Bot API.

## How to Run

### Backend
1. `cd backend`
2. `npm install`
3. Configure `.env` (Database credentials)
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
